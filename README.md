# Real Estate Market Trends Dashboard

## Project Overview

This Power BI dashboard was developed as part of the CodeAlpha Power BI Internship.

The project analyzes Hyderabad real estate market trends, property pricing, amenities, and location-based insights through interactive visualizations.

## Features

* Average Property Price Analysis
* Area and Bedroom Analysis
* Property Distribution by Location
* Real Estate Hotspot Identification
* Amenities Analysis
* Property Price vs Area Analysis
* Interactive Filters
* Geographic Visualization

## Tools Used

* Microsoft Power BI
* Power Query
* DAX
* CSV Dataset

## Key Insights

* Premium locations command significantly higher property prices.
* Property prices generally increase with larger area and bedroom count.
* High-demand locations were identified through hotspot analysis.
* Amenities such as Gym, Lift, Pool, and ClubHouse influence property attractiveness.
